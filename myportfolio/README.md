# Jan Allysandra Espia - Portfolio

Welcome to my personal portfolio repository! This website showcases my projects, experience, skills, and background as a Frontend Developer, GUI Editor, and Layout Artist.

## 💻 Tech Stack
- **HTML5**: Semantic and accessible markup.
- **CSS3**: Custom properties, responsive layouts, glassmorphism, and custom animations.
- **JavaScript (Vanilla)**: DOM manipulation, smooth scrolling, IntersectionObserver for animations, and Web Share API.
- **No Frameworks**: Built completely natively without any frameworks for maximum performance.

## ✨ Features
- **Dynamic Content**: All portfolio content is dynamically rendered from a single `data/portfolio.json` file.
- **Theme Toggle**: Built-in Light and Dark mode, with user preference saved in the browser.
- **Modern UI**: Features a sleek floating frosted-glass navigation bar and premium interface elements.
- **Native Sharing**: Integrated Web Share API to easily share the portfolio link on supported devices.
- **Responsive Design**: Fully optimized for seamless viewing on mobile, tablet, and desktop screens.

## 🛠️ Local Development
To run this project locally on your machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/Allysandra01/portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd portfolio
   ```
3. Because the site uses a `fetch()` call to load data from the JSON file, you need to run it through a local server. You can use VS Code's **Live Server** extension, or run:
   ```bash
   npx serve .
   ```
4. Open the provided localhost link in your browser!

## 📝 Customization
If you'd like to use this template:
1. Open `data/portfolio.json`.
2. Edit the text and links with your own personal details, experience, skills, and projects.
3. Swap out the images in the `css/images/` directory.
